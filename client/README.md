# CalorieDetect Pro - AI Molecular Calorie Detector

An ultra-premium, modern, and fully responsive biochemical calorie detection and nutritional analysis interface designed with HSL color theory, custom typography, glassmorphic aesthetics, and dynamic visualizations.

## Key Features
* **AI Scan Engine**: Comprehensive molecular ingredient scanner for nutritional biomarkers.
* **Hydration Cylinder**: A beautiful 3D glass water tracker with simulated liquid fluid mechanics.
* **Biometric Calculator**: Real-time interactive BMI status report indicator.
* **Diet Logbook**: Advanced daily log tracking and single-click reports compiled in PDF formats.
* **Comparison Matrix**: Side-by-side molecular calorie and macronutrient comparison dashboards.

## Technology Stack
* **Frontend**: Vanilla HTML5, Vanilla JavaScript (ESM modules), Custom CSS3 design tokens (Plus Jakarta Sans + Outfit Typography pair).
* **Charts**: Lightweight, responsive custom configurations powered by Chart.js.
* **Build System**: Clean, instant local bundling configurations.

## Getting Started
1. Install dependencies in both server and client folders: `npm install`
2. Launch the developer environment: `npm run dev`
